# Pet Mutation Lab Simulator

A unique Roblox genetics game where players mutate and combine pets in a lab to create new species, unlock upgrades, battle bosses, and monetize with premium features.

## Features
- Gene system: color, size, ability, personality
- Mutation chamber: interactive sliders for mutation
- Pet personalities: affect behavior and lab interactions
- Lab upgrades: unlock new features and speed up progress
- Boss battles: defeat bosses with genetically matched pets
- Monetization: gamepasses, premium starter pack, exclusive genes

## Getting Started
1. Import scripts and folders into Roblox Studio.
2. Replace asset placeholders in `Assets` with your own models and UI.
3. Set up gamepasses and monetization in Roblox.
4. Customize and expand features as desired.

## Premium Starter Pack
- Includes a powerful pet with rare genes and abilities
- Unlocks extra mutation slots, faster mutation times, and exclusive gene types


## Folder Structure
- `Scripts/` - Core game logic
- `UI/` - User interface scripts
- `Assets/` - Placeholder for models, textures, sounds
- `Monetization/` - Gamepass and premium logic
- `LabUpgrades/` - Upgrade shop scripts
- `Bosses/` - Boss battle logic
